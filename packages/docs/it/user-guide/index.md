# Guida utente

<kbd>⌘</kbd> = <kbd>Command</kbd> (<kbd>Ctrl</kbd> su Windows/Linux), <kbd>⌥</kbd> = <kbd>Option</kbd> (<kbd>Alt</kbd>), <kbd>⇧</kbd> = <kbd>Shift</kbd>.
:::
