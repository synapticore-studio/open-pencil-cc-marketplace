# Навигация по холсту

Холст — это ваше бесконечное рабочее пространство. Вы можете свободно перемещаться и масштабировать для навигации по дизайну.

## Перемещение

Перемещение видимой области холста без влияния на объекты.

- **Space + перетаскивание** — удерживайте Space и перетаскивайте в любом месте холста
- **Перетаскивание средней кнопкой мыши** — нажмите и перетаскивайте средней кнопкой мыши
- **Двумя пальцами на трекпаде** — проведите двумя пальцами по трекпаду

## Инструмент «Рука»

Нажмите <kbd>H</kbd>, чтобы активировать инструмент «Рука» для непрерывного перемещения. Любое перетаскивание на холсте сдвигает область просмотра без необходимости удерживать Space. Переключитесь на другой инструмент (например, <kbd>V</kbd> для выделения), чтобы деактивировать.

## Масштабирование

Увеличение и уменьшение масштаба относительно позиции курсора.

- <kbd>Ctrl</kbd> + прокрутка (или <kbd>⌘</kbd> + прокрутка на Mac) — прокрутка вверх для увеличения, вниз для уменьшения
- **Жест сведения пальцев** — сведите пальцы на трекпаде для масштабирования
- **Сочетания клавиш** — см. таблицу ниже

Жест сведения пальцев на панелях UI (слои, свойства) блокируется, чтобы случайно не изменить масштаб браузера.

## Сочетания клавиш

| Действие | Mac | Windows / Linux |
|----------|-----|-----------------|
| Перемещение | <kbd>Space</kbd> + перетаскивание | Space + перетаскивание |
| Инструмент «Рука» | <kbd>H</kbd> | <kbd>H</kbd> |
| Увеличить | <kbd>⌘</kbd><kbd>+</kbd> | <kbd>Ctrl</kbd> + <kbd>+</kbd> |
| Уменьшить | <kbd>⌘</kbd><kbd>−</kbd> | <kbd>Ctrl</kbd> + <kbd>−</kbd> |
| Масштаб 100% | <kbd>⌘</kbd><kbd>0</kbd> | <kbd>Ctrl</kbd> + <kbd>0</kbd> |

## Советы

- Масштабирование всегда направлено на позицию курсора, поэтому наводите указатель на то, что хотите рассмотреть ближе.
- Инструмент «Рука» полезен, когда нужно часто перемещаться — он остаётся активным, пока вы не переключите инструмент.
- См. [Выделение и манипуляции](./selection-and-manipulation) для работы с объектами на холсте.
