# Экспорт

Экспортируйте отдельные узлы как изображения или сохраняйте и открывайте целые документы в формате .fig.

## Экспорт изображений

Выделите узел и используйте раздел «Экспорт» в панели свойств.

### Настройки экспорта

- **Масштаб** — 0.5×, 0.75×, 1×, 1.5×, 2×, 3×, или 4× (скрыт для SVG — векторная графика не зависит от разрешения)
- **Формат** — PNG (прозрачный фон), JPG (белый фон), WEBP (прозрачный фон), SVG (вектор)

Вы можете добавить несколько настроек экспорта, чтобы экспортировать один и тот же узел в разных масштабах или форматах за один раз. Предпросмотр с шахматным фоном показывает, что будет экспортировано.

### Методы экспорта

| Метод | Mac | Windows / Linux |
|-------|-----|-----------------|
| Сочетание клавиш | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>E</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>E</kbd> |
| Контекстное меню | Правый клик <kbd>→</kbd> Экспорт… | Правый клик <kbd>→</kbd> Экспорт… |
| Панель свойств | Нажать кнопку «Экспорт» | Нажать кнопку «Экспорт» |

Экспортированный файл сохраняется через нативный диалог (настольное приложение) или скачивание в браузере.

## Копировать/Вставить как

Помимо экспорта в файл, вы можете скопировать выделение в буфер обмена в нескольких форматах через контекстное меню (правый клик → Копировать/Вставить как):

| Действие | Сочетание (Mac) | Сочетание (Win/Linux) |
|----------|-----------------|----------------------|
| Копировать как текст | — | — |
| Копировать как SVG | — | — |
| Копировать как PNG | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>C</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>C</kbd> |
| Копировать как JSX | — | — |

- **Копировать как текст** — копирует видимое текстовое содержимое из выделения
- **Копировать как SVG** — копирует выделение как SVG-разметку (вставляйте в редакторы кода, Inkscape и т.д.)
- **Копировать как PNG** — рендерит в масштабе 2× и копирует в буфер обмена (готово для вставки в Slack, Notion и т.д.)
- **Копировать как JSX** — копирует представление OpenPencil JSX (совместимо с `renderJsx()`)

## Операции с файлами .fig

OpenPencil использует формат .fig для полных документов — тот же бинарный формат, что и Figma.

### Открытие файлов

| Действие | Mac | Windows / Linux |
|----------|-----|-----------------|
| Открыть файл | <kbd>⌘</kbd><kbd>O</kbd> | <kbd>Ctrl</kbd> + <kbd>O</kbd> |

Открывается диалог выбора файла с фильтром по файлам .fig. В настольном приложении используется нативный диалог ОС.

### Сохранение файлов

| Действие | Mac | Windows / Linux |
|----------|-----|-----------------|
| Сохранить | <kbd>⌘</kbd><kbd>S</kbd> | <kbd>Ctrl</kbd> + <kbd>S</kbd> |
| Сохранить как | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>S</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>S</kbd> |

- **Сохранить** — перезаписывает текущий открытый файл без диалога
- **Сохранить как** — открывает диалог сохранения для выбора нового местоположения

Сохранённые файлы сжимаются и включают миниатюру для предпросмотра.

### Совместимость двустороннего обмена

Файлы, экспортированные из OpenPencil, можно открывать в Figma, и наоборот. Формат .fig сохраняет все типы узлов, свойства, заливки, обводки, эффекты, векторные данные и настройки раскладки.

## Сочетания клавиш

| Действие | Mac | Windows / Linux |
|----------|-----|-----------------|
| Экспорт выделения | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>E</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>E</kbd> |
| Копировать как PNG | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>C</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>C</kbd> |
| Открыть файл | <kbd>⌘</kbd><kbd>O</kbd> | <kbd>Ctrl</kbd> + <kbd>O</kbd> |

| Сохранить как | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>S</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>S</kbd> |

## Советы

- Используйте масштаб 2× или 3× при экспорте для экранов с высоким разрешением.
- JPG всегда использует белый фон — используйте PNG или WEBP, если нужна прозрачность.
- Используйте экспорт в SVG, когда нужен векторный формат для дальнейшего редактирования в Illustrator, Inkscape или коде.
- Миниатюра в экспортированных файлах .fig позволяет просматривать файлы в файловых менеджерах и в выборе файлов Figma.
