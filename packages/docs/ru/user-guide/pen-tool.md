# Инструмент «Перо»

Инструмент «Перо» создаёт векторные контуры, используя модель данных векторной сети, совместимую с форматом .fig Figma.

## Активация

Нажмите <kbd>P</kbd>, чтобы активировать инструмент «Перо».

## Расстановка точек

- **Клик** — ставит угловую точку (прямолинейный сегмент)
- **Клик + перетаскивание** — ставит точку кривой с касательными ручками Безье: направление и длина перетаскивания определяют форму кривой

Кликайте по нескольким точкам, чтобы строить контур сегмент за сегментом. Линия предпросмотра тянется от последней поставленной точки к вашему курсору при перемещении.

## Замыкание контура

Кликните по **первой точке** контура, чтобы замкнуть его в петлю. Замкнутые контуры можно заливать.

## Открытые контуры

Нажмите <kbd>Escape</kbd>, чтобы зафиксировать текущий контур как открытый. Открытые контуры отображаются только обводкой — заливка не применяется.

## Векторные сети

Контуры используют модель данных векторной сети вместо простых списков точек. Векторные сети позволяют создавать более гибкую топологию — например, ветвящиеся контуры — и полностью совместимы с форматом .fig.

## Сочетания клавиш

| Действие | Mac | Windows / Linux |
|----------|-----|-----------------|
| Инструмент «Перо» | <kbd>P</kbd> | <kbd>P</kbd> |
| Зафиксировать открытый контур | <kbd>Escape</kbd> | Escape |

## Советы

- Линия предпросмотра всегда начинается от последней поставленной точки — она не перескакивает на (0,0).
- Перетаскивайте дальше при постановке точки кривой, чтобы сделать кривую шире.
- После создания контура используйте панель свойств для настройки заливки, обводки и эффектов.
