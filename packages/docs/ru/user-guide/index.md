# Руководство пользователя

OpenPencil — это редактор дизайна с открытым исходным кодом, совместимый с Figma. Полностью локальный, с нативной поддержкой ИИ и возможностью программирования. Это руководство охватывает всё, что нужно знать для эффективной работы с редактором.

::: tip Кроссплатформенные сочетания клавиш
В этом руководстве сочетания клавиш приведены в нотации Mac: <kbd>⌘</kbd> = <kbd>Command</kbd> (<kbd>Ctrl</kbd> на Windows/Linux), <kbd>⌥</kbd> = <kbd>Option</kbd> (<kbd>Alt</kbd>), <kbd>⇧</kbd> = <kbd>Shift</kbd>.
:::
