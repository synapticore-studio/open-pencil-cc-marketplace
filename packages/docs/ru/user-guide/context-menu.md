# Контекстное меню

Кликните правой кнопкой мыши по холсту, чтобы открыть контекстное меню. При клике правой кнопкой по узлу он сначала выделяется. Клик правой кнопкой по пустому холсту снимает выделение.

## Копировать/Вставить как

Подменю **Копировать/Вставить как** предлагает дополнительные форматы буфера обмена для выделенных узлов:

| Действие | Сочетание (Mac) | Сочетание (Win/Linux) |
|----------|-----------------|----------------------|
| Копировать как текст | — | — |
| Копировать как SVG | — | — |
| Копировать как PNG | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>C</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>C</kbd> |
| Копировать как JSX | — | — |

- **Копировать как текст** — копирует видимое текстовое содержимое из выделения
- **Копировать как SVG** — копирует дерево узлов как SVG-разметку
- **Копировать как PNG** — рендерит в масштабе 2× и помещает в системный буфер обмена (вставляйте в Slack, Notion и т.д.)
- **Копировать как JSX** — копирует представление OpenPencil JSX для использования с `renderJsx()`

## Действия с буфером обмена

| Действие | Сочетание (Mac) | Сочетание (Win/Linux) |
|----------|-----------------|----------------------|
| Копировать | <kbd>⌘</kbd><kbd>C</kbd> | <kbd>Ctrl</kbd> + <kbd>C</kbd> |
| Вырезать | <kbd>⌘</kbd><kbd>X</kbd> | <kbd>Ctrl</kbd> + <kbd>X</kbd> |
| Вставить сюда | <kbd>⌘</kbd><kbd>V</kbd> | <kbd>Ctrl</kbd> + <kbd>V</kbd> |
| Дублировать | <kbd>⌘</kbd><kbd>D</kbd> | <kbd>Ctrl</kbd> + <kbd>D</kbd> |
| Удалить | ⌫ | <kbd>Backspace</kbd> / Delete |

Действия с буфером обмена недоступны, когда ничего не выделено (кроме «Вставить», которое доступно при наличии содержимого в буфере обмена).

## Z-порядок

| Действие | Сочетание |
|----------|-----------|
| На передний план | ] |
| На задний план | [ |

Перемещает выделенный узел в начало или конец списка дочерних элементов родителя.

## Группировка

| Действие | Сочетание (Mac) | Сочетание (Win/Linux) |
|----------|-----------------|----------------------|
| Сгруппировать | <kbd>⌘</kbd><kbd>G</kbd> | <kbd>Ctrl</kbd> + <kbd>G</kbd> |
| Разгруппировать | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>G</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>G</kbd> |
| Добавить авто-раскладку | <kbd>⇧</kbd><kbd>A</kbd> | <kbd>Shift</kbd> + <kbd>A</kbd> |

- **Сгруппировать** требует 2 или более выделенных узла
- **Разгруппировать** появляется при выделении группы — дочерние элементы перемещаются к родителю группы
- **Добавить авто-раскладку** оборачивает выделение в новый фрейм с [авто-раскладкой](./auto-layout)

## Действия с компонентами

Действия с компонентами отображаются фиолетовым цветом в соответствии с цветовой темой компонентов.

| Действие | Сочетание (Mac) | Сочетание (Win/Linux) | Доступно для |
|----------|-----------------|----------------------|--------------|
| Создать компонент | <kbd>⌥</kbd><kbd>⌘</kbd><kbd>K</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>K</kbd> | Фреймы, группы, множественное выделение |
| Создать набор компонентов | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>K</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>K</kbd> | 2+ выделенных компонента |
| Создать экземпляр | — | — | Компоненты (без сочетания клавиш) |
| Перейти к основному компоненту | — | — | Экземпляры |
| Отсоединить экземпляр | <kbd>⌥</kbd><kbd>⌘</kbd><kbd>B</kbd> | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>B</kbd> | Экземпляры |

См. [Компоненты](./components) для подробностей о работе с компонентами.

## Видимость и блокировка

| Действие | Сочетание (Mac) | Сочетание (Win/Linux) |
|----------|-----------------|----------------------|
| Скрыть / Показать | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>H</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>H</kbd> |
| Заблокировать / Разблокировать | <kbd>⇧</kbd><kbd>⌘</kbd><kbd>L</kbd> | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>L</kbd> |

Метка переключается в зависимости от текущего состояния узла (например, «Скрыть» для видимого узла, «Показать» для скрытого).

## Переместить на страницу

Подменю **Переместить на страницу** показывает все страницы, кроме текущей. Выберите страницу, чтобы переместить выделенные узлы под холст этой страницы.

## Советы

- Клик правой кнопкой по пустому холсту даёт доступ к «Вставить» — полезно для размещения содержимого в определённом месте.
- Действия с компонентами появляются только когда это уместно (например, «Создать экземпляр» только для узлов-компонентов).
- Контекстное меню отражает сочетания клавиш — это хороший способ узнать неизвестные вам сочетания.
